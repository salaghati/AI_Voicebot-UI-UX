## Overview

This directory is the container for the multi-step "new campaign" flow under docs/src/app/(console)/bot-engine/campaigns. There are no root-level files; instead the flow is organized into four step subdirectories (step-1 through step-4), each of which contains a single route-level TypeScript React page component (page.tsx). Each step directory's page.tsx implements the UI and behavior for that particular step in the campaign creation flow.

Together, the step-1..step-4 subdirectories represent the sequential screens that make up the campaign creation user journey. step-1 contains the first-step page component and is described as the first step of the flow at the route level. step-2 provides the second-step page component. step-3 and step-4 each contain a single page.tsx implementing their respective screens. The directory's role in the larger system is to group the route-level page components for the new campaign flow within the bot-engine campaigns area so the application routing and navigation can treat these steps as distinct pages in the multi-step flow.


### File Organization

No files at the root; the directory is organized into one subdirectory per step (step-1, step-2, step-3, step-4). Each subdirectory contains a single route-level TypeScript React page component (page.tsx) that implements that step's UI. This keeps each step isolated and easy to locate.
